## About The Project

Telegram Bot made for a bakery. Users can see products availability schedule, delivery times, make orders. Admin can manage products, billets, orders.

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* PHP v8.x.

### Installation

<ol>
	<li>Upload to web-hosting.</li>
	<li>Create database from a sample file.</li>
	<li>Create your Telegram bot at BotFather.</li>
	<li>Activate a webhook directed to your web-hosting.</li>
	<li>Type "Hello" to your new bot.</li>
</ol>
